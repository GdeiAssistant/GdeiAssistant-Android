# 广东二师助手安卓客户端

**广东二师助手**是一款旨在为广东第二师范学院在校师生提供最便捷获取校园生活、社团活动、教务信息的校园应用，提供了课表查询、成绩查询、四六级考试成绩查询、校园卡充值挂失和消费查询等综合性的服务与功能。

## 功能

- 成绩查询
- 课表查询
- 四六级查询
- 一键评教
- 消费查询
- 校园卡充值
